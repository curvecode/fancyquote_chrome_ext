# Chrome Extension Icons

## Icon Requirements

Chrome extensions require multiple icon sizes for different contexts:

### Required Sizes:
- **16px**: Extension management page, favicon
- **24px**: Action icon (on some displays) 
- **32px**: Action icon, Windows
- **48px**: Extension management page, notifications
- **128px**: Chrome Web Store, installation

### How to Replace Icons:

1. **Design your icon** in your preferred graphics software
2. **Export as PNG** in all required sizes (16, 24, 32, 48, 128px)
3. **Save in the icons/ directory** with exact names:
   - icons/icon16.png
   - icons/icon24.png  
   - icons/icon32.png
   - icons/icon48.png
   - icons/icon128.png
